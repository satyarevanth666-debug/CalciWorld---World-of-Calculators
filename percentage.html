Percentage Calculator Guide

The percentage calculator helps you quickly solve common math questions such as "What is 20% of 500?" or "How much did a value change by?" It is useful for finance, grades, discounts, and everyday comparisons.

What this calculator does

It computes simple percentage values and can be used to calculate percentage change, proportionate values, and related ratios.

Formula used

There are two main formulas:

  • Percentage of a value: Result = (percentage ÷ 100) × value
  • Percentage change: Change% = ((new − old) ÷ old) × 100

Step-by-step instructions

  1. Enter the value you want to analyze.
  2. Enter the percentage you want to apply.
  3. Click calculate to see the result.
  4. For percentage change, enter the old and new values if the calculator supports it.

Worked examples

Example 1: Discount calculation

Find 15% of ₹2,000.

Result = 0.15 × 2,000 = ₹300.

Example 2: Grade conversion

Find 87% of 50 marks.

Result = 0.87 × 50 = 43.5 marks.

Use cases

  • Calculate discounts during shopping.
  • Estimate tax or tip amounts.
  • Convert grades and test scores.
  • Measure percentage change in prices or values.

FAQs

Can I use it for percentage increase or decrease?

Yes, use the percentage change formula to calculate increases or decreases between two values.

Do I need a calculator for basic percent problems?

This tool makes it faster and reduces error, especially with non-round numbers.

How to use this page

Use the percentage calculator for any problem involving ratios, discounts, or changes in value. Check your work with the examples above.
